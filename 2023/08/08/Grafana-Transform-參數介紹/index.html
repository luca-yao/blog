Grafana-Transform-參數介紹

2023-08-08
Monitor
681 詞
  • Grafana

進入正題

建好圖表後，可以在編輯edit中的，Transform中使用

用途 說明
Add field from calculation 使用現有字段計算成新的類別
Concatenate fields 將所有字段串聯起來組成新字段 ( 在Table View 下才看的出來)
Config from query results 將查詢結果中提許字段，合併到另外一個查詢中
Convert field type 將字段轉換成其他類別
Create heatmap 根據資料計算熱圖
Extract fields 針對資料再分析(JSON，Labels)
Field lookup 透過外部DNS查詢數據來源
Filter by name 透過正規表示式或標籤來篩選數據
Filter data by query 透過群組來過濾數據
Filter data by values 透過數值來判斷篩選方式
Group by 按照指定的標籤做分組，並且對每個組別數據做處理
Grouping to matrix 將三個字段組成一個matrix
Histogram 畫長條圖
Join by field 將多個表之間相同的字串，合併到一個表中
Join by labels 將Labels的結果展示到Table中
Labels to fields 按照時間分組排列，並返回標籤或標記於table中
Limit 限制顯示的資料量
Merge 將多個查詢合併成一個table
Organize fields 重新排序、隱藏、重新命名字串/列，只能用於當前的圖表
Partition by values 透過一個或多個字段中的唯一值進行區分
Prepare time series 將時間序列數據從寬格式轉換為長格式或反之亦然
Reduce 壓縮整合字段
Rename by regex 使用正則表達式和替換模式重命名部分查詢結果
Rows to fields 行到字段轉換將行轉換為單獨的字段
Series to rows 將多個時間序列數據查詢的結果合併為一個結果
Sort by 排序
Spatial operations 將空間操作應用於查詢結果中

以上只是簡單的說明一下個功能的用途，實際在操作的時候會有多是需要互相搭配使用的

如果想要更清晰簡單的理解，可以去看看 Grafana Labs 的 文件說明，裡面也有舉些簡單的案例。

網址在這邊：https://grafana.com/docs/grafana/latest/panels-visualizations/query-transform-data/transform-data/#series-to-rows
